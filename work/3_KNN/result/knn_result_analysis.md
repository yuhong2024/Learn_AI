# KNN实验结果分析

## 1. K值与准确率关系

实验通过遍历不同的K值（1~20），在测试集上评估KNN模型的分类准确率，并绘制了K-准确率曲线。

- **K-准确率曲线（k_acc_curve.png）**  
  ![K-准确率曲线](k_acc_curve.png)

- **数据表（k_acc_curve.csv）**  
  K=3时准确率达到最高，为1.0000。

**分析：**  
从曲线可以看出，随着K值的增加，模型的准确率先上升后趋于平稳。K=3时模型在测试集上表现最佳，准确率达到100%。K过小易受噪声影响，K过大则可能掩盖局部特征。
